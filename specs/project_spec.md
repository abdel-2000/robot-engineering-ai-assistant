# Robot Engineering AI Assistant

## Project Goal

Develop a multi-agent AI assistant that helps robotics engineers design, understand and develop robotics software safely.

---

## Mission

Help robotics developers by:

- Exploring robotics projects
- Understanding project architecture
- Explaining source code
- Generating documentation
- Checking safety constraints
- Suggesting software improvements

---

## Target Users

- Robotics Engineers
- AI Engineers
- ROS2 Developers
- Students
- Researchers

---

## Main Features

### Explorer

Analyze robotics projects and explain their structure.

### Documentation

Generate technical documentation.

### Safety

Review robot actions and suggest safe operating conditions.

### Architecture

Explain how robotics software is organized.

---

## Technologies

- Python
- Google ADK
- Gemini
- MCP
- ROS2 (generic)
- MQTT (generic)

---

## Out of Scope

This project DOES NOT:

- Control a real robot
- Use confidential company information
- Use proprietary SDKs
- Use internal documentation
- Execute dangerous robot actions

---

## Project Philosophy

Safety First

Human Approval

Reusable AI Agents

Modular Skills

Generic Robotics Development
